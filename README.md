# QuickOrder - Smart QR Ordering for Restaurants

Transform your restaurant with QuickOrder. An easy-to-use digital menu and ordering system that works right in your customer's mobile browser.

## Features

- **Smart QR Ordering**: Customers can scan and order in seconds.
- **No App Required**: Works instantly on any smartphone browser.
- **Modern Digital Menu**: Showcase your food with a beautiful, responsive interface.
- **Improved Efficiency**: Reduce order errors and speed up service by 40%.

## Getting Started

1. **Clone the repository**:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Run the development server**:
   ```sh
   npm run dev
   ```

## Technology Stack
